1. Inštalacia django do python:  
pip install django   
pip list

2. vytvorenie django projektu: 
    django-admin startproject nazov_projektu .

3. nastavenie slovenčiny pre projekt(mysite/settings.py):
    LANGUAGE_CODE = 'sk'
    TIME_ZONE = 'Europe/Bratislava'

4. spustenie vyvojoveho serveru
    python manage.py runserver  

5. vytvorenie aplikacie 
     python manage.py startapp nazov_aplikacie

6. zaregstrovanie aplikacie do projektu(mysite/settings)     
    INSTALLED_APPS = [
        'django.contrib.admin',
        'django.contrib.auth',
        'django.contrib.contenttypes',
        'django.contrib.sessions',
        'django.contrib.messages',
        'django.contrib.staticfiles',
        'nazov_aplikacie',
    ]

7. vytvorenie jednoduchého views (skuska/views.py)

    return HttpResponse('Hello world')

8. vytvorenie urlpattern:
8.1 mysite/urls.py
    from django.contrib import admin
    from django.urls import path, include 

urlpatterns = [
    path('admin/', admin.site.urls),
    path('skuska/', include('skuska.urls')),
]

from django.urls import path
from . import views

    path('', views.index, name='index' ),

9. Template
9.1 
- vytvoriť v priečinku  aplikacie priečinok templates/nazov_aplikacie
- v tomto priečinku vytvoriť html súbor (napr. index.html)
9.2 upraviť views tak , aby použival template
 
from django.shortcuts import render, HttpResponse
from . import views

def index(request):
    sucet = 10 + 5
    sucin = 10 * 5
    apar = "cislo A je parne
    return render(request, 'kalkulacka/index.html', {"sucet":sucet, "sucin":sucin, "apar":apar})

10.css , obrazky 
- vytvoriť v priečinku  aplikacie priečinok static/nazov_aplikacie
- v tomto priečinku vytvoriť css (napriklad kalkulacka.css)
- do tohto priečinku nakopirovat obrazky  
- do template (index.html) doplniť: 
    - uplne na začiatok {% load static %}
    - do hlavičky odkaz na style 
    <head>
        <link rel="stylesheet" href="{% static 'kalkulacka/kalkulacka.css' %}">
    </head>
- obrazok vložiť:
<img src={% static 'kalkulacka/kalkulacka.png' %}>
