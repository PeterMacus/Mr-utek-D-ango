1. Inštalacia django do python:  
pip install django   
pip list

2. vytvorenie django projektu: 
    django-admin startproject nazov_projektu .

3. nastavenie slovenčiny pre projekt(mysite/settings.py):
    LANGUAGE_CODE = 'sk'
    TIME_ZONE = 'Europe/Bratislava'

4. spustenie vyvojoveho serveru
    python manage.py runserver  

5. vytvorenie aplikacie 
     python manage.py startapp nazov_aplikacie

6. zaregstrovanie aplikacie do projektu(mysite/settings)     
    INSTALLED_APPS = [
        'django.contrib.admin',
        'django.contrib.auth',
        'django.contrib.contenttypes',
        'django.contrib.sessions',
        'django.contrib.messages',
        'django.contrib.staticfiles',
        'nazov_aplikacie',
    ]

7. vytvorenie jednoduchého views (skuska/views.py)
from django.shortcuts import render, HttpResponse

def index(request):
    return HttpResponse('Hello world')

8. vytvorenie urlpattern:
8.1 mysite/urls.py
    from django.contrib import admin
    from django.urls import path, include 

urlpatterns = [
    path('admin/', admin.site.urls),
    path('skuska/', include('skuska.urls')),
]

8.2
vytvoriť súbor urls.py v priečubku aplikacie(skuska.py)
upraviť urls.py: 
from django.urls import path
from . import views

    path('', views.index, name='index' ),
